# Project 1 - Travel Fun

## EventBrite API
Search The Events API by using:
* categories
* date range
* latitude
* longitude
* distance

Returns the events within the searched parameters with:
* name
* description (if available)
* Event URL
* Time (start and end)
* Capacity
* Image (if available)
